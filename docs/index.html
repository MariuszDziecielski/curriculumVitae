Dlaczego IT?

Z mojego punktu widzenia najważniejsze czynniki, które decydują o atrakcyjności branży IT to możliwość kontaktu i pracy z fajnymi, kreatywnymi ludźmi – ludźmi, od których mogę się czegoś nowego nauczyć a także perspektywa stałego rozwoju i doskonalenia swoich umiejętności.

Ciągły rozwój i podnoszenie kwalifikacji oznacza również nowe wyzwania oraz brak monotonii i rutyny czyli rzeczy, których każdy chciałby uniknąć, gdyż powodują one brak satysfakcji z wykonywanego zajęcia.

    Gra napisana w Angular (TypeScript) i stworzona w oparciu o grid Bootstrap'a. Domyślne style frameworka zostały zmodyfikowane dla potrzeb projektu. Gra jest osadzona w uniwersum Star Treka a jej zasady są takie same jak w znanej od wieków popularnej grze "Papier, kamień i nożyce".
    Live / GitHub

Umiejętności / technologie

  • JavaScript - skrypty zgodne z OOP i z wykorzystaniem ES6, debugowanie przy pomocy narzędzi developerskich w Chrome
  • ES6 - klasy, rozbudowane właściwości obiektu, ulepszona obsługa parametrów, funkcje strzałki, const i let, szablony literałów
  • JQuery - operacje na elementach DOM, obsługa zdarzeń i proste animacje
  • Angular - Angular CLI, moduły i komponenty, wiązania danych, strumienie obserwowalne, usługi, routing, DI, komunikacja między komponentami
  • TypeScript - typy podstawowe, sparametryzowane i generyczne, klasy i modyfikatory dostępu, moduły, interfejsy i dekoratory
  • RxJS - praktyczne wykorzystanie strumieni obserwowalnych - tworzenie, subskrypcja i jej anulowanie, operatory.
  • Ajax - pobieranie danych z serwera i korzystywanie z API
  • Grunt - instalacja i podstawowa konfiguracja zadań do automatyzacji
  • Git - konfiguracja środowiska pracy i praktyczne korzystanie z repozytoriów
  • HTML5 - cięcie z psd i kodowanie w oparciu o dostarczony projekt graficzny
  • CSS3 - zaawansowane selektory i pseudoselektory, przejścia, animacje oraz transformacje na obiektach
  • Sass - korzystanie z funkcji typu zagnieżdżanie selektorów, mixiny oraz zmienne
  • RWD - tworzenie responsywnych stron w oparciu o flexbox i grid oraz media queries
  • Bootstrap - praktyczne wykorzystanie i modyfikacje dla potrzeb danego projektu
